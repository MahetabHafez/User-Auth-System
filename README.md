# 🚀 Full-Stack User Authentication System

A modern, secure, and responsive user authentication system with a custom dashboard. Built using the **MERN** stack logic (Node.js, Express, MongoDB) and featuring a sleek **Glassmorphism** UI.

## ✨ Features
* 🔐 **Secure Auth:** User Registration and Login with hashed passwords using `Bcrypt`.
* 🎫 **JWT Protection:** Secure access to dashboard using JSON Web Tokens.
* 💻 **Interactive Dashboard:** Personalized user area displaying profile information.
* 🎨 **Glassmorphism UI:** Modern dark-themed design with smooth CSS animations.
* 🌐 **Live Deployment:** Hosted on Vercel with a cloud-connected MongoDB Atlas database.

## 🛠️ Tech Stack
* **Backend:** Node.js, Express.js
* **Database:** MongoDB Atlas (Mongoose)
* **Authentication:** JWT (JSON Web Tokens)
* **Frontend:** HTML5, CSS3 (Glassmorphism), JavaScript (Vanilla)

## ⚙️ Local Installation
If you want to run this project locally:

1. Clone the repository:
   ```bash
   git clone [https://github.com/MahetabHafez/User-Auth-System.git](https://github.com/MahetabHafez/User-Auth-System.git)
2. Install dependencies:
   ```bash
   npm install
3.Create a .env file and add your credentials:
  ```bash 
   MONGO_URI=your_mongodb_uri 
   JWT_SECRET=your_secret_key 
   PORT=5000

4.Start the server:
   ```bash
node server.js
